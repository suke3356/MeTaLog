MeTaLog について

これは何？

MeTaLog は、勉強メモ・思考ログ・知識整理のための個人用メモアプリです。

フォルダ、タグ、検索、Markdown、バックアップ、オフライン起動に対応しています。

主な機能

  • フォルダで分類
  • タグで横断整理
  • Markdown で読みやすく記述
  • 検索で素早く探す
  • インポート / エクスポートでバックアップ
  • PWAとしてホーム画面から起動
  • オフラインでも使用可能

使い方

  1. 📁 フォルダ追加 で科目や分類を作る
  2. フォルダを開いて 📝 ログ追加
  3. 本文中に #タグ名 を書く
  4. 上のタグ一覧や検索から後で見返す

タグの例

#電験 #理論 #リアクタンス #要復習

保存について

データは端末内の IndexedDB に保存されます。

Safari / Chrome のデータ削除で消える可能性があるため、定期的にエクスポートでバックアップするのがおすすめです。
